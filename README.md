# project_angkot_vision

Project scaffold for detecting and validating "angkot" using vision + LLM.

Structure:

```
project_angkot_vision/
├── config/
├── data/
├── notebooks/
├── src/
├── scripts/
├── .env
├── .gitignore
├── requirements.txt
└── README.md
```

Follow the `config/config.yaml` for paths and thresholds. Keep `.env` out of git.
